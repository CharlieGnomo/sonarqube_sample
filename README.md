# pytest-example
pytest - Python Testing Example

This is a simple example of how to use pytest to integreate testing in your Python application.


##Code Structure

```
myapp/
    app.py
    mymodule/
        __init__.py
        funcs.py
tests/
    __init__.py
    test_app.py
        mymodule/
            __init__.py
            test_additions.py
            test_funcs.py
```
